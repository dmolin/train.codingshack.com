What is this course all about?

Front-End Development is an area of Web Development that includes all of the skills required to create the visual and interactive part of a Website. It's a complex mix of skills and technologies and one of the most fast growing and well paid job roles of the moment.

The Front-End Developer translates a graphic design document (created by a graphic designer) into a live and interactive Web page. The Front End Developer brings the application to life creating rich, compelling experiences and allowing the user to interact with the data behind the application. It’s a trending job, engaging, well paid and full of challenges and wonders.

This course will teach you the skills to kickstart a career in Front End Development, assuming no previous knowledge of any topic.

What will I learn taking this course?

We'll touch on all the foundational topics that form the toolbelt of a professional Front-End Developer:

  • Semantic HTML
  • CSS
  • Responsive Web Development
  • Javascript, Ajax and a dash of jQuery
  • Backbone.js a robust MVC Framework!
  • The art of Unit Testing (brownie points during an Interview!)

Semantic HTML

HTML is the language used to build web pages; Thanks to HTML, a textual document gets translated into styled text, images, videos, links, interactive forms. We'll learn how to describe content using Semantic HTML.

CSS

CSS is the language used for describing the visual look and formatting of an HTML document. We'll learn how to use CSS and in our Workshop we'll create a beautifully looking website, starting from a Photoshop design document, using HTML and CSS!

Javascript

Javascript is a programming language created for the Web. It is designed to "run" in a web browser. We'll learn how to use it to create HTML and make our page interactive!

jQuery

We'll learn how to harness the power of jQuery, the most famous Javascript library in the web, to create and manipulate HTML through exercises and examples.

Backbone.js

We'll learn how to use Backbone.js, a powerful Javascript Framework, to create professional Rich Web Applications, putting together Ajax, jQuery, Underscore and all the technologies seen so far.

Mocha - Unit Testing Framework

We'll transition from amateurs to serious professionals learning how to properly test our Javascript code, using the testing framework Mocha!
